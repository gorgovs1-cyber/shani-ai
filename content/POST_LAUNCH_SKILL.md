## הגדרת ManyChat לכל פוסט

### Flow לכל פוסט חדש:

**Step 1 — Comment Trigger:**
- Trigger: תגובה מכילה [TRIGGER_WORD]
- Instagram Post: הפוסט הספציפי (לא כל הפוסטים)

**Step 2 — Public Comment Reply:**
```
שלחתי לך את [שם הריסורס] בהודעות 📩
```

**Step 3 — Check Condition:**
- Is Following = Yes → המשך ל-Step 4
- Is Following = No → שלח DM "לא עוקב"

**Step 4 — DM (עוקב):**
```
היי! [שם הריסורס] שלך מוכן 🤍

לחצי כאן להורדה:
👉 [קישור ל-PDF / Google Drive]

נהנית? תגידי לי בתגובות 💬
```

**Step 5 — DM (לא עוקב):**
```
היי! כדי לקבל את [שם הריסורס] קודם עשי לי עוקב:
👉 instagram.com/shani.creates.ai

אחרי שעקבת — הגיבי שוב [TRIGGER_WORD] בפוסט 🤍
```

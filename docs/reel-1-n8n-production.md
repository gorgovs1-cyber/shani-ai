## שלב 0 — הכנה (פעם אחת, ~15 דק')

כדי שהקלטת המסך תהיה אמיתית:
1. ב-Claude → פרויקט חדש בשם **"אוטומציות n8n"**.
2. הדביקי את הוראות הפרויקט (מתוך המדריך `n8n-claude-automations.html`).
3. העלי 3 דוגמאות JSON מ-n8n.io/workflows.
4. פתחי חשבון n8n חינמי (n8n.io).
5. תרגלי פעם אחת: פרומפט → JSON → Import. זה מה שתצלמי.

## שלב 1 — הקול (ElevenLabs)

1. ElevenLabs → Voice → **Instant Voice Clone** → העלי 1–2 דק' הקלטה שלך בעברית (טון רגוע, ברור). שמרי.
2. לכל שורת קריינות בטבלה (קטעים 1,2,3,4,5,6,7): הדביקי → בחרי את הקול שלך → Generate → הורידי MP3.
   - שמרי כל קטע בנפרד: `voice-1.mp3`, `voice-2.mp3` וכו'.

## שלב 2 — האווטאר (Higgsfield Lip-Sync Studio)

רק לקטעים המדברים: **1, 2, 3, 7** (לא לקטעי הקלטת המסך).
1. Higgsfield → **Lip-Sync Studio** (או Kling AI Avatar).
2. העלי **תמונת פנים חזיתית שלך** (`shani-about.jpg` — מולמצלמה, לא פרופיל) + ה-MP3 של אותו קטע.
3. Generate → קליפ אווטאר מדבר בקול שלך. חזרי לכל קטע מדבר.
   - יחס: 9:16. שמרי: `avatar-1.mp4`, `avatar-2.mp4`...

## שלב 3 — הקלטות מסך (Win+G)

3 קליפים קצרים (10–15ש' כל אחד), לאט וברור:
- **קליפ Claude A:** מקלידה פרומפט בעברית: *"כל ליד חדש מטופס האתר — הוסף לגיליון Google ושלח לי התראה בוואטסאפ."*
- **קליפ Claude B:** Claude מייצר JSON (גלילה שמראה שזה ארוך ואמיתי).
- **קליפ n8n C:** תפריט 3 נקודות → Import from Clipboard → האוטומציה נוחתת על הקנבס.

מדריך התקנה · Claude Code

לתת ל-Claude לצפות
בסרטוני וידאו

התקנה מלאה, שלב-אחר-שלב, של הסקיל /watch — שמאפשר ל-Claude להוריד ריל מאינסטגרם / טיקטוק / יוטיוב, לצפות בו באמת, לתמלל ולנתח אותו סצנה-אחר-סצנה. מושלם לפירוק ריילסים של מתחרים.

⏱ ~15 דקות
רמת קושי · בינונית
עלות · חינם
Windows · PowerShell
לפני שמתחילים

מה צריך להכין

שלושה דברים, וכולם מסתדרים תוך כדי. אל תדאגו אם משהו חסר — המדריך מתקין הכל.

🧩

הדרישות

1 · Node.js מותקן במחשב  ·  2 · חשבון Claude  ·  3 · להיות מחוברים לאינסטגרם בדפדפן Chrome (הכלי מוריד דרך העוגיות שלכם).

התהליך

9 שלבים להתקנה

1

בדקו ש-Node.js מותקן

פותחים Windows PowerShell ומריצים את השורה. אם חוזר מספר גרסה (כמו v24.16.0) — מצוין. אם שגיאה — מורידים מ-nodejs.org ומתקינים.

PowerShell
node --version
2

התקינו את Claude Code

הכלי שבו הכל רץ. ההתקנה לוקחת דקה — ירוצו שורות טקסט, זה תקין.

PowerShell
npm install -g @anthropic-ai/claude-code
3

הפעילו את Claude Code

מקלידים claude. בפעם הראשונה יתבקשו להתחבר לחשבון, לאשר את התיקייה, ולאשר שימוש בדפדפן Chrome — אשרו את השימוש בדפדפן (זה מה שמאפשר את ההורדה מאינסטגרם). המסך ישתנה — עכשיו אתם בתוך Claude Code.

PowerShell
claude
4

הוסיפו את המרקטפלייס

מכאן והלאה הפקודות מתחילות ב-/ ורצות בתוך Claude Code — לא ב-PowerShell.

Claude Code
/plugin marketplace add bradautomates/claude-video
5

התקינו את הסקיל

בוחרים "Install for you (user scope)" — כך הוא זמין בכל הפרויקטים.

Claude Code
/plugin install watch@claude-video
6

הפעילו מחדש

מרעננים כדי שהסקיל ייכנס לפעולה.

Claude Code
/reload-plugins
7

מריצים — וההתקנה האוטומטית משלימה את החסר

בהרצה הראשונה הסקיל בודק אם חסרים yt-dlp ו-ffmpeg (מורידים ומעבדים את הווידאו). אם חסר — בחרו התקנה דרך winget, וזה מסתדר לבד.

Claude Code
/watch <קישור-לריל> <השאלה שלך>
# בחירת התקנה: winget install yt-dlp.yt-dlp
8

מפתח Groq לתמלול (חינם)

כדי שהכלי גם יתמלל את הדיבור, מוסיפים מפתח חינמי. נכנסים ל-console.groq.com, נרשמים (בלי כרטיס אשראי), Create API Key, מעתיקים את המפתח (gsk_…) ומדביקים כשמתבקש. בוחרים אפשרות Groq ואז Other כדי להדביק.

🔒

המפתח פרטי

המפתח מוצג פעם אחת בלבד — העתיקו מיד. אל תשתפו אותו בפומבי; אפשר תמיד ליצור חדש.

9

בוחרים רמת פירוט — ומוכנים

לניתוח ריילסים בחרו balanced (מומלץ) — פריימים מודעי-סצנה עד 100. וזהו, ההגדרה הושלמה. מהרגע הזה פשוט קוראים לכלי:

Claude Code
/watch https://instagram.com/reel/XXXX מה ההוק? תפרקי מבנה, קצב וקאטים
מה עושים עם זה

לפרק ריילסים של מתחרים בשניות

מדביקים קישור לריל של מתחרה ומבקשים ניתוח: מה ההוק ב-3 השניות הראשונות, סדר הסצנות, כל כמה שניות יש קאט, מה ה-CTA — ומה כדאי לקחת מזה לתוכן שלכם. מה שלקח לצפות ולנתח ידנית, קורה עכשיו אוטומטית.
